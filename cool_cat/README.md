# Крутой Кот

Настольное приложение “Крутой Кот”: слева изображение кота, справа — панель управления для кормления, поглаживания и озвучивания текста. UI на русском.

## Почему PySide6
Выбран **PySide6 (Qt)** как основной GUI-фреймворк: он кроссплатформенный, современный и лучше подходит для настольных приложений. Это соответствует предпочтительному варианту из ТЗ.

## ДЕФОЛТЫ (можно поменять)
- Стартовая сытость: `60`.
- Порог голода: `40`.
- Уменьшение сытости: `-1` каждые `10` секунд.
- Кормление: `+20` (не выше 100).
- Поле ввода — **однострочное**.

Все значения хранятся в `config.py`.

## Как работает
- Если кот **сытый**, он озвучивает введённый текст выбранным голосом.
- Если кот **голодный**, он отказывается и грубо отвечает.
- “Покормить” повышает сытость.
- “Погладить” возвращает реплику (разные варианты для сытого/голодного).
- Любая озвучка попадает в лог.

## Озвучка (TTS)
- На macOS используется системная команда `say`.
- Для русского выбирается голос **Yuri** (если доступен), для английского — **Alex**.
- Если голос не найден, используется системный по умолчанию и показывается предупреждение в UI.
- На Windows/Linux — заглушка (предупреждение в UI), чтобы не ломать macOS.


Если файла `assets/cat.png` нет, будет показан текст “Нет изображения”.
